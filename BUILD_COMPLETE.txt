================================================================================
    QUICK START
================================================================================

1. Install Dependencies:
   cd /Users/cope/Passbook_Oracle/microservices/flora-mercury-service
   npm install

2. Configure Environment:
   cp .env.example .env
   # Edit .env with your settings

3. Generate Encryption Key:
   node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
   # Add to .env as ENCRYPTION_KEY

4. Start Service:
   npm run dev    # Development with auto-reload
   npm start      # Production

5. Verify:
   curl http://localhost:3020/health

================================================================================
    ENVIRONMENT VARIABLES
================================================================================

Required:
  MONGODB_URI        - MongoDB connection string
  JWT_SECRET         - JWT secret (must match main Flora app)
  ENCRYPTION_KEY     - 32-byte hex key for AES-256-GCM

Optional:
  PORT=3020          - Server port
  NODE_ENV           - Environment (development/production)
  LOG_LEVEL=info     - Winston log level
  MOCK_MODE=true     - Enable mock mode
  MERCURY_API_KEY    - Mercury API credentials (not needed in mock)
  MERCURY_API_URL    - Mercury API base URL
